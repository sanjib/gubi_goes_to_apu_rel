# Gubi Goes to APU (and Saves the World)

<img width="1143" height="626" alt="image" src="https://github.com/user-attachments/assets/b4f4f743-edcc-4036-b22b-e89210cb0338" />

This is the release repository for a prototype game as part of the Masters of Science in AI, Final Year Project.

To play the game, go to the Release page and download the file "gubi_goes_to_apu.zip".
It's a Windows only release. Unzip the folder, then run game.bat.
